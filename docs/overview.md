## システムの目的

個人事業主・小規模法人向けに、領収書の処理を自動化するAIエージェントシステム。カスタムGPTに領収書を送るだけで、OCR抽出・勘定科目判定・Google Sheets登録・銀行照合までを自動で行う。

---

## システム構成

    [ユーザー]
        ↓ 領収書・指示
    [カスタムGPT（フロントエンド）]
        ↓ OpenAI Actions（HTTP POST）
    [Google Apps Script（バックエンド）]
        ↓
    [Google Drive]  [Google Sheets]
        領収書保存      経費データ管理

## 主要機能一覧

| 機能 | 概要 | 優先度 |
|:--|:--|:--|
| 領収書アップロード | 最大10件を一括処理 | P0 |

| 外貨自動換算 | 77銀行仲値をOpenAI経由で取得 | P0 |
| 銀行照合 | GMO・MF・通帳写真と自動照合 | P0 |
| AIレビューモード | 月末に needsReview案件を一括提示 | P1 |
| 全体レビュー | 横断的異常検知（重複・月次異常等） | P1 |
| 定期経費自動入力 | 家賃等を月末に自動登録 | P1 |
| 交通費自動計算 | OpenAI経由でIC運賃を取得・キャッシュ管理 | P1 |
| フィードバック学習 | ユーザー指摘をルールとして学習 | P2 |

## 重要な設計決定

主な設計決定の詳細は [docs/decisions/](decisions/) フォルダを参照。

| 決定事項 | 結論 |
|:--|:--|
| 貸方科目 | 照合前は短期借入金、法人口座照合後は普通預金 |
| 論理削除 | 物理削除禁止、isDeletedフラグで管理 |
| シート書き込み | appendRow_ / updateRow_ 経由を強制 |
| 為替レートURL | buildFxRateUrl_() で動的生成 |
| MF照合方式 | ホワイトリスト＋照合済みIDの組み合わせ |
| 照合ステータス | 5段階管理（unreconciled / reconciled / reconciled_modified / cash_payment / reconcile_pending） |
| 月割り方式 | 前払費用計上（短期前払費用特例は顧問税理士に確認） |
| レビュートリガー | デフォルトは月末（Script Propertiesで変更可） |
